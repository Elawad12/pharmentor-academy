# الخطة النهائية والمضمونة لرفع المشروع (حل مشكلة الصلاحيات 403)

يا صديقي، ألف مبروك! لقد كتبت الأمر بشكل صحيح تمامًا في المحاولة السابقة، ووصلنا للعقبة الأخيرة. رسالة `Permission denied (403)` تعني أن مفتاحك ليس لديه صلاحية "الكتابة" في المستودع.

هذه هي الخطة النهائية لحل المشكلة 100% عن طريق إنشاء مفتاح جديد بالصلاحيات الصحيحة.

---

### الخطوة 1: حذف المفتاح القديم (مستحسن)

1.  اذهب إلى [صفحة مفاتيح GitHub](https://github.com/settings/tokens). (أنت بالفعل في هذه الصفحة).
2.  ابحث عن المفتاح القديم الذي كنت تستخدمه (اسمه `PharMentor`) واضغط على زر `Delete` الأحمر بجانبه.

---

### الخطوة 2: إنشاء مفتاح جديد بالصلاحيات الصحيحة (الأهم)

1.  في نفس الصفحة، اضغط على `Generate new token` واختر **`Generate new token (classic)`**.

2.  **اسم المفتاح:** في خانة `Note`، اكتب اسمًا للمفتاح، مثلاً: `PharMentor-Studio-Key`.

3.  **تاريخ الانتهاء:** في قسم `Expiration`، اختر `90 days`.

4.  **الصلاحيات (أهم خطوة):** انزل إلى قسم `Select scopes`. **يجب عليك تفعيل الخيار الرئيسي `repo`**.
    *   **ضع علامة صح (✓) في المربع بجانب كلمة `repo`**.
    *   عندما تفعل ذلك، سيتم تفعيل كل الخيارات الفرعية تحته تلقائيًا. هذا هو ما يعطي المفتاح صلاحية رفع الكود.

5.  انزل لأسفل الصفحة واضغط على الزر الأخضر `Generate token`.

6.  **انسخ المفتاح الجديد فورًا!** سيظهر لك المفتاح (يبدأ بـ `ghp_`). اضغط على أيقونة النسخ بجانبه واحفظه في مكان آمن مؤقتًا.

---

### الخطوة 3: تجهيز المشروع (للتأكيد فقط)

إذا قمت للتو بتنفيذ هذه الأوامر ونجحت، يمكنك تخطيها. إذا لم تكن متأكدًا، نفذها مرة أخرى **أمرًا بأمر**.

```bash
rm -rf .git
```
```bash
git init
```
```bash
git add .
```
```bash
git commit -m "Final project setup"
```
```bash
git branch -M main
```

---

### الخطوة 4: استخدام الأمر الصحيح مع المفتاح الجديد

1.  خذ هذا القالب الفارغ وانسخه إلى محرر نصوص:
    ```bash
    git push --set-upstream https://Elawad12:PASTE_YOUR_NEW_TOKEN_HERE@github.com/Elawad12/pharmentor-academy.git main
    ```

2.  **بدقة شديدة،** احذف `PASTE_YOUR_NEW_TOKEN_HERE`.

3.  **فورًا مكانها وبدون أي مسافات**، ألصق المفتاح **الجديد** الذي نسخته للتو.

4.  انسخ الأمر الكامل والمُعدل من محرر النصوص، ألصقه في الطرفية، ثم اضغط `Enter`.

هذه هي الطريقة الكاملة والدقيقة. أنا واثق تمامًا أنها ستنجح هذه المرة.
